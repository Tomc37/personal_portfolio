Number Guesser

One of the first projects completed as part of the course. This is a simple guessing game against a computer player with random numbers generated as the target number and for the computer's guess. Whoever is closest to the target number gains a point, the player getting the point in case of a tie. The html elements were all provided but the script file had to be completed from ground up to get the game working. You can find the game here and the github respository here.

Mysterious Organism

This was one of the first major JavaScript projects undertaken. The idea was to build an object factory in JavaScript that creates 15 random DNA pairs of a mysterious organism. Pairing the key as the organisms specimen number along with the value of DNA pairs. After that, additional functions were added as methods to use certain criteria to judge the organisms survival and force mutations on individual DNA bases. The file itself is just a .js script file and would need to be ran in terminal or console. You can find the github repository here.
