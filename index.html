Number Guesser

One of the first projects completed as part of the course. This is a simple guessing game against a computer player with random numbers generated as the target number and for the computer's guess. Whoever is closest to the target number gains a point, the player getting the point in case of a tie. The html elements were all provided but the script file had to be completed from ground up to get the game working. You can find the game here and the github respository here.

Mysterious Organism

This was one of the first major JavaScript projects undertaken. The idea was to build an object factory in JavaScript that creates 15 random DNA pairs of a mysterious organism. Pairing the key as the organisms specimen number along with the value of DNA pairs. After that, additional functions were added as methods to use certain criteria to judge the organisms survival and force mutations on individual DNA bases. The file itself is just a .js script file and would need to be ran in terminal or console. You can find the github repository here.

Find Your Hat Game

After building out more skills in JavaScript, I moved on to then make an interactive game. Using an array to create a 2D map, we have our character moving around this map using simple commands in order to find their "hat". This is the game's winning condition but the game can be lost if the character falls down a hole or attempts to move out of bounds of the map. You can find the gitub repository here but please note that the game requires some familiarity with Node JS to be ran.

Jammming

Following the Find Hat project, I then moved on to develop skills in the areas of asynchronous JavaScript, HTTP requests and the React framework. This resulted in the Jammming web-app that you can find here.
This web-app communicates with the Spotify API, requiring login to a Spotify account as an alternate method of creating playlists on your spotify account. You simply search for songs and add them to a playlist and save that playlist.
All of the components are built using React class components with several states being passed to props with data being received from and transmitted through the Spotify API.
